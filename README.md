# DiscourseReply

Chrome 浏览器插件，在 Discourse 论坛的 Boost 按钮旁添加快速回复功能。

## 功能

- 💬 一键回复 Boost 内容
- 📋 自动引用原帖内容
- @ 自动提及原作者

## 安装

### 从 Release 下载

1. 前往 [Releases](https://github.com/chenhuawang-04/DiscourseReply/releases) 下载最新版本
2. 解压压缩包
3. 打开 Chrome，访问 `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」，选择解压后的文件夹

### 开发模式

```bash
git clone https://github.com/chenhuawang-04/DiscourseReply.git
cd DiscourseReply
```

然后按上述步骤 3-5 加载插件。

## 使用

1. 访问任意 Discourse 论坛帖子
2. 在 Boost 气泡旁会出现 💬 回复图标
3. 点击图标即可引用内容并 @用户回复
